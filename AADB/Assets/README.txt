Leaderboard example.

* Uses XDK Online/Stats example SPA configuration.
* To use your own title configuration, replace the title id and SPA configuration with your own.

1. Build XDK Stats sample - %XEDK%\Source\Samples\Online\Stats\
2. Rename Debug/spa.bin to Stats.spa and copy to the project folder (%XEDK%\Source\Samples\Online\Stats\)
3. Select Stats.spa in Player Publishing Settings.
4. Check the "Generate _SPAConfig.cs" box. A new file should appear in your Project window.
5. Set the title id to FFFF006A in Player Publishing Settings.
